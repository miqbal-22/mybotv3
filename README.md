## Install
Follow The Steps Below!

```bash
> termux-setup-storage
(after that tap on permission)
> pkg update
> pkg upgrade
> pkg install wget
> pkg install ffmpeg
> pkg install nodejs
> pkg install tesseract
> pkg install git -y
> git clone https://github.com/miqbal-22/mybotv3
> cd mybotv3
> bash install.sh
```

## Usage

```bash
> npm start
```
